# Bank_Transaction_API using Flask and MongoDB

#### Requirements
--------------
  Python3
  MongoDB
  flask
  Bcrypt
  pymongo
  flask_restful
  
## API Endpoints
    `/register`
     `/add`
     `transfer`
     `balance`
     `takeloan`
     `payloan`
     
## Using the API
  `git clone https://github.com/surajkamblesk/Bank_Transaction_API.git`
  
  `pip install -r requirements.txt `
